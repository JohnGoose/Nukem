# Doom

3D game engine made with C. This game is computed on the CPU only.

**Keys:**
* `W | S  - Move forward and backwards`
* `Tab.   - Activate Mouse`
* `Q      - Main Menu`
* `A | D  - Strafe left/right`
* `ESC    - Quit`
